# Сценарий демонстрации для Дархана

## Подготовка
- Ссылка: https://school-management-mu-one.vercel.app
- Логин: admin@school.kz / admin123
- Данные: расписание на неделю 13.04, посещаемость за 13-15.04, 18 оплат

---

## Часть 1 — Вход (2 мин)

**Что делать:** Открыть ссылку → ввести логин/пароль → войти

**Что говорить:**
> "Дархан, вот система. Каждый пользователь заходит по логину/паролю. У Айнуры будет роль Администратор — полный доступ. У тебя — Руководитель, видишь аналитику и финансы. У педагогов — только своё расписание и ЗП."

**Показать:** sidebar слева — 15 разделов, внизу имя и роль + кнопка Выйти

---

## Часть 2 — Расписание (10 мин)

**Что делать:** Нажать "Расписание" → неделя 13.04

**Что говорить:**
> "Вот расписание на неделю. Сетка: по горизонтали педагоги, по вертикали время. Цвет = тип занятия. Есть два режима: Пн/Ср/Пт и Вт/Чт."

**Показать кнопки:**
1. "Импорт из Google Sheets" → "Айнура загружает расписание из своей Google таблицы. Один раз вставила ссылку — система всё распределила по педагогам и кабинетам."
2. "Продлить" → нажать → "Выбрала 4 недели — расписание скопировалось на месяц вперёд. Не надо заново вбивать."
3. "Заморозка" → показать диалог → "Каникулы с 1 по 15 июня — нажала, все занятия в этом периоде отменились автоматически."
4. "Копировать с пред. недели" → "Или просто скопировать прошлую неделю."

**Кликнуть на ячейку** → показать диалог создания слота вручную

---

## Часть 3 — Посещаемость (5 мин)

**Что делать:** Нажать "Посещаемость" → дата 13.04.2026

**Что говорить:**
> "Конец рабочего дня. Айнура открывает посещаемость и отмечает каждого ребёнка."

**Показать:**
1. Статистика вверху: "Был: 15, Отсутствует: 2, Больничный: 2"
2. Карточка урока — "09:00 Середа Оксана, индивидуальное"
3. **Кликнуть на статус ученика** — показать как переключается: Был → Без причины → Уважительная → Больничный → Перенос → Отработка
4. Кнопка "Замена" → "Педагог заболел — Айнура нажала Замена, выбрала другого. ЗП автоматически пересчитается."
5. Кнопка "Ассистент" → "Второй педагог на уроке — отдельная ставка."
6. Легенда: "6 статусов. Каждый влияет на оплату. Был = родитель платит, педагог получает. Без причины = родитель платит ШТРАФ, педагог НЕ получает."

**Методический час** — показать вверху: "Методисты отмечаются отдельно — состоялся/не состоялся."

**Что делать:** "Ученики" → кликнуть на фамилию любого ученика

**Что говорить:**
> "Полная карточка ребёнка. Вверху: тариф, посещаемость в процентах, начислено, долг."

**Показать табы:**
1. "Информация" — тарифная история, перерасчёты
2. "Посещения" — все визиты за 3 месяца с цветными статусами
3. "Оплаты" — история оплат
4. "Заморозки" — больничные, каникулы
5. "CRM" → "Тут Айнура записывает: позвонила маме, обещала оплатить 15 числа. Всё в одном месте."

**Кнопки:** "Заморозка" и "Перерасчёт" — показать диалоги
